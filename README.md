# LSTM_based_nonstationary_multiponts_ground_motions
# Nonstationary Spatial Coherence Modeling via WPT and Deep Recurrent Neural Networks

This repository contains the official code and data processing scripts for the paper:
**"Modeling Nonstationary Seismic Spatial Coherence Using Wavelet Packet Transform and Deep Recurrent Neural Networks"** *(Currently under review at CACIE)*

## 📌 Overview
This repository provides a novel, data-driven computational paradigm to model the nonstationary lagged coherence of spatially variable seismic ground motions. Unlike traditional empirical models that assume stationarity, this framework captures the dynamic temporal evolution of spatial coherence using sequence learning and physical wave-propagation mechanics.

## 🚀 Key Features (Modules)
 **Deep Sequence Learning (Ensemble LSTM):** Implements a Deep Ensembles Long Short-Term Memory (LSTM) architecture with an innovative intensity-frequency domain transformation (using the normalized envelope).
